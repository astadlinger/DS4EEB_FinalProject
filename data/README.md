# What is in this folder?
## Rodent_activity.csv
### csv file that contains data on rodent observations (camera, duration, etc.)
## Seed_predation.csv
### csv file that contains data on seed predation success at each plot and conditions of the plot
## Trapping.csv
### csv file that contains data on specific camera traps
## Vegetation_cover.csv
### csv file that contains data on conditions on specific plots
## edi files and manifest.txt
### raw dataset
